## Building MEVN Project from Scratch

### Install Node
```
sudo apt install nodejs
node -v
```

### Install Vue CLI
```
npm install -g @vue/cli
vue --version
```

### Setup a Vue Project
```
vue create shopsavvy
```

- Choose manual setup
- Choose TypeScript, VueRouter, ESLint

### Install packages
```
npm install --save express body-parser cors mongoose dotenv
```
